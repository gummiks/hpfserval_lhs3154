# HPF Spectra of LHS 3154
137 1D spectra of LHS 3154 from the Habitable-zone Planet Finder on the 10m Hobby-Eberly Telescope as discussed in Stefansson et al. 2023.

Precise radial velocities can be extracted from the spectra using the HPF-SERVAL code here: https://github.com/gummiks/hpfserval_lhs3154
